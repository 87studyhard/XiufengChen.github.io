Prototype in JavaScript

2016-04-26
Front End

原型

原型对象

我们创建的每个函数都有一个prototype（原型）属性，这个属性是一个指针，指向一个对象，而这个对象的用途是包含可以有特定类型的所有实例共享的属性和方法。意思是，不管创建实例1还是实例2，他们访问的都是同一组属性和方法。

无论什么时候，只要创建了一个新函数，就会根据一组特定的规则为该函数创建一个prototype属性，这个属性指向函数的原型对象。而原型对象包含一个constructor属性，指向原函数。
当调用构造函数创建一个新实例后，该实例的内部将包含一个指针[[Prototype]]（内部属性），指向构造函数的原型对象。Runtime可以使用这个内部属性构成的prototype-chain查找属性。一般情况下我们无法使用[[Prototype]]，但是有的JS runtime提供 __proto（后两个下划线省略）属性得到构造函数的原型，例如chrome。

所以，我们可以把构造函数看做类，原型的属性看做共享的变量，每个实例的方法实现方式相同，就把它加在prototype上。对象的字段是属于各个实例的，则在构造方法中添加。

Dynamic Programming

2016-03-24
Algorithm

本文来自于算法导论第十五章动态规划的学习。

分治法和动态规划法的区别是，分治法在于将问题划分为互不相交的子问题，递归地求解子问题，再将它们组合起来，求出原问题的解。与之相反，动态规划应用于子问题重叠的情况，即不同的子问题具有公共的子子问题（子问题的求解是递归进行的，将其划分为更小的子子问题）。动态规划算法对每个子子问题只求解一次，将其解保存在一个表格中，从而无需每次求解一个子子问题是都重新计算，避免了这种不必要的计算工作。

动态规划用来求解最优化问题，一般情况下，问题有多种解，但是要求出一个最优解，也许该问题有多个最优解。

How many types of database languages are?

There are four types of database languages:

  • Data Definition Language (DDL) e.g. CREATE, ALTER, DROP etc.
  • Data Manipulation Language (DML) e.g. SELECT, UPDATE, INSERT etc.
  • DATA Control Language (DCL) e.g. GRANT and REVOKE.
  • Transaction Control Language (TCL) e.g. COMMIT and ROLLBACK.
