BFS: 广度优先搜索

广度优先搜索二叉树（层序遍历）是用队列来实现的，从二叉树第一层（根节点）开始，自上之下逐层遍历，从左到右的顺序对结点逐一访问。

  1. 初始化一个队列，把根节点入队列；
    2.当队列为非空时，执行步骤三到步骤五，否则，执行步骤六；
    3.出队列取得一个结点，访问该节点；
    4.若出队列的左子树为非空，则将该节点的左子树入队列；
    5.若出队列的右子树为非空，则将该节点的右子树入队列；
    6.结束。

DFS：深度优先搜索实现方式：栈。

深度优先遍历，相当于先根遍历，采用非递归方法实现，需要辅助的数据结构是栈。
